# Mod 反向导入器

这是一个 Blender 插件，用于把已经由导出的 mod 文件夹重新导入为可编辑网格。有很多bug，但能用，自己跟AI battle就可以了。  

之所以上传这个脚本，无非是有个CN社区小丑罢了，又当又立，又卖Mod逆向又卖加密还卖防报错骗小白，并且还有一群死臭虫跟着这头出生，绝了。
